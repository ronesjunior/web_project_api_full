# Web Project API Full

### <u>FRONTEND:</u>

site: https://web-project-api-full-flax.vercel.app/

Aplicação frontend desenvolvida com React, integrada a uma API REST para gerenciamento de usuários e cards, com autenticação via JWT.

### Funcionalidades

- Cadastro e login de usuário
- Autenticação com JWT
- Proteção de rotas
- CRUD de cards
- Atualização de perfil
- Curtir e remover curtidas
- Interface responsiva

### Tecnologias Utilizadas

- **React**
- **React Router DOM**
- **Context API**
- **Fetch API**
- **JWT**
- **Vite**

---

### <u>BACKEND:</u>

API construída seguindo boas práticas:

### Recursos

- Estrutura MVC (Model - View - Controller)
- Autenticação e Autorização com JWT
- Hash de senha com bcrypt
- Validação com Celebrate/Joi
- Middleware de erro
- Middleware de logs de requisições e erros
- Deploy no Render

### Tecnologias Utilizadas

- **Node.js**
- **Express**
- **MongoDB**
- **Mongoose**
- **JWT**
- **bcryptjs**
- **CORS**
- **dotenv**
